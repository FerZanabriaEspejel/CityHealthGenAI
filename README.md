# Urban Health Risk Prediction using Generative AI

This project aims to develop a predictive and generative AI system to analyze and prevent urban health risks associated with mobility and environmental conditions.

## Objectives
- Predict air pollution and health risk levels
- Generate actionable recommendations
- Support urban decision-making

## Technologies
- Python
- Machine Learning (Random Forest, LSTM)
- Data Analysis (Pandas, NumPy)

## Structure
- data/: datasets
- notebooks/: experiments
- src/: core code
- results/: outputs
